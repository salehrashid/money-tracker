# Codex Session Rules

## Use New Chat For

- Project setup
- Firebase/database foundation
- Transactions
- Dashboard
- Statistics
- OCR
- Notification listener
- myBCA parser if notification chat became too long
- Final review

## Same Chat Is Allowed For

- Firebase foundation + database models if stable
- Transactions + search filter if stable
- Notification listener + myBCA parser if stable

## Warning Signs To Start New Chat

- Codex starts modifying unrelated files.
- Codex forgets the project structure.
- The diff becomes too large.
- The chat has many corrections.
- The feature scope becomes unclear.
